## 🌟 Features

- 🎨 **Bold Unicode Aesthetic** - Stunning visual design with neon glows and glass morphism
- ⚡ **Real-time Streaming** - Get AI responses as they generate
- 🌐 **Cloud AI Integration** - Access powerful models via Ollama Cloud
- 📱 **Responsive Design** - Works beautifully on desktop and mobile
- 🔐 **Secure API Handling** - API keys loaded safely from environment variables
- 🎯 **Model Selection** - Easy to configure different AI models
- 🛑 **Stream Abort** - Stop generation mid-stream
- 💾 **Chat History** - Keep track of your conversations
- 🌈 **Modern Animations** - Smooth, delightful interactions

## 🚀 Quick Start

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn
- Ollama Cloud API key ([Get one here](https://ollama.com/settings/keys))

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/ollama-chat-webclient.git
cd ollama-chat-webclient

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env

# 4. Edit .env and add your API key
# OLLAMA_API_KEY=your_api_key_here

# 5. Run the development server
npm run dev
